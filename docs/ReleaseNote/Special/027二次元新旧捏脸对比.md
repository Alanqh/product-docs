# 二次元新旧捏脸对比

## 总结：

**本次优化是在旧版基础上**

**1，增大了脸部轮廓相关调整项的可控范围。**

**2，五官可以更独立操作，五官的调整范围更合理。**

**3，对原有的面部区域进行了总结，明确区分下颌，颧骨，脸颊，下巴几个区域。使捏脸操作更具有可控性。**

**4，增大了下颌部分范围，可以支持更多的脸型。**
